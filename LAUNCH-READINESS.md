# K-RoadTrip 런칭 준비 체크리스트 (Launch-Readiness)

> `launch-reviewer`가 이 기준으로 검수하고, 운영자(나)가 진행 상황을 추적한다.
> **런칭은 binary** — 핵심(데이터·수익·지도·i18n) 미충족이면 런칭 불가.

## 1. 데이터 (현재 전부 임시 → 실제로)
- [ ] 모든 루트가 실제·현지검증 (더미 라우트 0)
- [ ] 모든 장소가 실재 + 좌표 정확 + 현지검증
- [ ] 더미/테스트 데이터 프로덕션에서 완전 제거

## 2. 미디어
- [ ] 모든 영상이 실제 한국 풍경 (샘플 영상 0)
- [ ] 미디어 자체 호스팅 (외부 샘플 사이트 의존 제거)
- [ ] 이미지·영상 최적화 (포맷·사이즈·lazy·poster)

## 3. 수익 Funnel (매출 직결) 💰
- [ ] 렌터카·항공 제휴 링크 실재 + 정상 동작
- [ ] 추적/레퍼럴 파라미터 보존 + 전환 귀속 확인
- [ ] 결제 경로 끝까지 테스트 (클릭 → 파트너 → 결제)
- [ ] 제휴 키·시크릿 클라이언트 노출 0
- [ ] 제휴 disclosure 명시 (신뢰 + 일부 법규 대응)

## 4. 핵심 UX — 지도 (앱 존재 이유) 🧭
- [ ] 네이버맵 원탭 핸드오프 동작 + 좌표 정확
- [ ] 지도/링크 실패 시 폴백
- [ ] 네이버맵 미설치 사용자 graceful 경로

## 5. i18n 🌐
- [ ] 영어 100% (한국어 누출 0)
- [ ] 한국어 토글 동작
- [ ] 미번역·하드코딩 문자열 0

## 6. 성능·모바일 📱
- [ ] 모바일 네이티브 (썸존·세로형 미디어)
- [ ] Lighthouse 성능 양호, LCP 관리
- [ ] 해외 접속 속도 확인

## 7. 신뢰·법무
- [ ] 메시지 일관성 (no-ads 주장 vs 스폰서 모순 해결)
- [ ] 제휴 disclosure
- [ ] 약관·개인정보 처리방침 (외국인 데이터 처리)

## 8. 인프라·운영(solo)
- [ ] 프로덕션 환경변수/시크릿 세팅 (개발과 분리)
- [ ] Supabase RLS 전체 ON
- [ ] 도메인·SSL
- [ ] 모니터링 (다운·폼·제휴 클릭 감지)
- [ ] 데이터 갱신이 혼자서 가능한 구조
