# K-Riders 트래픽 획득 전략 (2026-07-02 확정)

> 근거: `docs/research/research-global-apps.md`(REVER·Kurviger·Calimoto·iOverlander 심층)
> + `docs/research/research-korean-communities.md`(바튜매·산뽕·라이트바겐·내비 생태계).
> 목표: 네이버 카페(바튜매 87만, 산뽕오프로드)와 라이트바겐 사용자층에서 트래픽을 가져온다.

## 1. 핵심 결론 3줄

1. **킬러 피처 = "이륜차 안전 루트"** — 한국은 OECD 유일 이륜차 고속도로+자동차전용도로
   전면금지 국가(위반=형사처벌)인데 내비 3사 이륜차 모드가 전부 불완전(카카오 재탐색 버그,
   티맵 V9 격하, 네이버 오안내). 커뮤니티 GPS 트랙 = "이륜차가 실제 간 길"의 증거 데이터.
2. **카페 직접 홍보는 자살행위**(바튜매 아웃링크 영구강퇴 사례) — 유일한 침투법은
   **회원이 자발적으로 퍼가고 싶은 아티팩트**(코스 카드 이미지·GPX·와인딩 지수)를 만들어
   회원 손으로 들어가게 하는 것. 주 채널은 오픈웹 SEO("와인딩 코스 추천" 상위가 지도·GPX
   없는 텍스트 블로그뿐 — 구조적 공백) > 유튜버 씨딩(최대어 16.7만, 씨딩 단가 저렴) >
   오픈채팅/밴드(링크 공유가 자연스러운 유일한 공간, 카톡 OG 미리보기가 훅).
3. **라이트바겐은 경쟁자가 아니다** — 거래·금융 플랫폼(2024 매출 162억·첫 흑자)이고
   코스/커뮤니티는 구색. 루트 기록·소셜 영역은 **지배자 없는 공백 시장**(유일 유사품:
   신생 인디앱 RideOn, 리뷰 10개). 단 라이트바겐 자금력(누적 143억)이 내려올 수 있으니
   소셜 그래프·루트 데이터 선점 속도가 중요.

## 2. 이번 스프린트(P1) — 전부 무료, 즉시 구현

| # | 기능 | 근거 | 구현 |
|---|---|---|---|
| A | **이륜차 안전 루트 배지** | §1-1 킬러 페인포인트. v1=작성자 자기선언("자동차전용도로/고속도로를 지나지 않아요") + GPS 트랙 실주행 근거 | `moto_safe` 컬럼 + 저장 폼 체크 + 눈에 띄는 배지 |
| B | **와인딩 지수 (K-Calimeter)** | Calimoto 최대 중독 포인트 = 재미점수 하나. 자체 방위각 계산 수십 줄, 라이선스 청정 | 트랙 저장 시 Σ\|Δbearing\|/km → 0~100 + 등급 라벨. 피드 정렬 추가 |
| C | **루트 유형 태그** | 산뽕(임도/오프로드) 세그먼트 진입로 + 피드 탐색성 | `route_type`(winding/touring/offroad/city) + 피드 필터 칩 |
| D | **공유 4종 세트** (트래픽 코어) | 3사 공통 성장엔진 = 공개 SEO 페이지+OG. 카톡/카페 미리보기가 곧 코스 카드 | 루트 상세 SSR 전환 + generateMetadata + **동적 OG 이미지(루트 궤적 그려진 카드)** + 공유 버튼 + GPX 다운로드(로그인 게이트=가입 트리거) |

## 3. P2 (다음 스프린트 후보)

- **라운드트립 자동 생성** — "출발지+거리 → 원형 코스": OpenRouteService 무료 키(일 2,500req)의
  `round_trip` 옵션. 국내 전무 기능. *사장님 액션: ORS 무료 키 발급.*
- **한국 라이더 POI 레이어** — Overpass로 주유소·바이크정비·전망대 시드(자체 DB 캐시, ODbL
  어트리뷰션) + iOverlander식 검수 큐·"최근 확인" 신선도·Warning(단속/노면) 카테고리.
  ⚠️ iOverlander 데이터 자체는 비CC — 절대 재사용 금지, 운영 모델만 벤치마킹.
- **임도 아카이브(산뽕 공략)** — 좌표 공개에 민감한 문화 확인됨 → 공개 피드보다
  **비공개 크루 공유(링크 초대)가 먼저**. 개방/폐쇄 상태(산불조심기간 2~5월) 표시.
- **그룹 위치공유 링크** — Supabase Realtime 재사용(채팅 자산과 시너지). REVER LiveRIDE 대체.
- **Open-Meteo 날씨/고도**(비상업 일 1만콜 무료, CC-BY) — 코스 고도 프로파일 + 출발 시간대 날씨.
- **임베드 위젯** — 네이버 블로그/티스토리 라이더 블로거 생태계용.

## 4. P3 (돈/인프라 필요 — 틀만, 가짜 UI 금지)

- **커브 강도 3단 라우팅(빠른길/와인딩/헤어핀)** — GraphHopper OSS(Apache 2.0)에 curvature
  커스텀 모델 내장(Kurviger와 동일 계보). 남한 OSM이면 1~2GB RAM VM(월 $5~10)으로 자가호스팅.
  *유료 인프라라 보류 — P2의 ORS로 수요 검증 후 전환.*
- 자동 자동차전용도로 검증(트랙 vs OSM motorroad 태그 대조), GPX 임포트, 연간 결산 리포트
  (12월 시즌오프 리텐션), B2B 브랜드 챌린지(REVER 모델 — 국내 용품샵 제휴).

## 5. 하지 말 것 (리서치로 확정)

- 오프라인 지도·음성 내비(웹 제약 + 네이버 핸드오프로 충분), 크래시 감지(4사도 없음),
  OSRM 데모서버 프로덕션 사용(정책 위반), OpenTopoMap 의존(생존 모드), iOverlander 데이터
  재사용(라이선스), GPL 라이브러리 프론트 번들(leaflet-elevation 등 — 직접 구현으로 대체),
  **네이버 카페 직접 홍보(영구강퇴)**.

## 6. 운영 플레이북 (코드 밖 — 사장님 영역)

- **SEO 콘텐츠 시드**: 검증된 명코스(양평~가평 중미산, 화천 아흔아홉굽잇길, 충주호반,
  국도35호선, 남해안 일주 등)를 운영자 계정으로 직접 기록/등록 → "OO 와인딩 코스" 검색 공략.
- **유튜버 씨딩**: 류석(16.7만)·솔리드레이어(9.2만) 등 탑 3~5명 — 코스 영상 설명란에
  "이 코스 지도/GPX" 링크. 시장이 작아 탑티어만 잡아도 씬 전체 도달.
- **오픈채팅/밴드**: 크루방에 "이번 주 코스" 루트 링크 — 카톡 OG 미리보기(D 기능)가 전제.
- **시즌 캘린더**: 3~11월 시즌, 2월 말 "시즌온 준비", 12월 "연간 주행 결산", 소음단속
  (105dB·음향카메라)·산불철 임도 폐쇄 정보 = 재방문 훅.
